##  Nasıl Çalıştırılır?

Bu alıştırmaları çalıştırmak için yerel bilgisayarınızda Python kurulu olmalıdır.

1.  Depoyu klonlayın:
    ```bash
    git clone [https://github.com/AylinOZEN/python-alistirmalar.git](https://github.com/AylinOZEN/python-alistirmalar.git)
    ```
2.  İstediğiniz dosyayı (örneğin `daire_alan_ve_cevre_hesaplama.py`) terminalde çalıştırın:
    ```bash
    python daire_alan_ve_cevre_hesaplama.py
    ```
